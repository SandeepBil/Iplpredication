# 🏏 IPL Match Winner Predictor

A single-file web app that predicts IPL match winners using real historical data from 1,175 IPL matches (2008–2026).

## Features
- 10 teams with current **IPL 2026 squads**
- Player stats filtered to **2021–2026 only**
- Head-to-head records with match history
- Venue stats for 14 major IPL grounds
- Toss, pitch type & day/night factors
- Chase difficulty calculator
- Win probability model with feature breakdown

## How to use
Just open `ipl_predictor.html` in any browser — no install, no server, no dependencies.
